*연산자

-이항 연산자(binary operator)
: 피연산자가 둘인 연산자
: 연산자마다 우선 순위와 결합 방향이 다르다. (계산하는 순서 존재)
>대입, 산술, 복합, 관계(대소), 논리 연산자 등이 존재
>SCE, short circuit evaluation
: 연산의 효율 및 속도를 높이기 위해서 불필요한 연산을 자동으로 생략하는 행위
: 예시
// false && 이면, 뒤에 뭐가 와도 false이기 때문에 뒤에 연산을 실행하지 않는다.
// num2 = 0 이고, false && num2 += 10 이라는 문장이 존재하면 num2는 여전히 0이다.
// true || 도 마찬가지이다.

-단항 연산자
: 피연산자가 하나인 연산자
>부호 연산자 : +, -
>증가 및 감소 연산자: ++, --

-비트 연산자
: 정수에서만 지원되며 비트 단위로 연산한다
> &: and, |: or, ^: XOR, ~: 비트 반전
: 넘어가고 다음에 필요하면 공부 ㅋ

-비트 쉬프트 연산자(<< , >> , >>>)
: 여기도 필요할 때 공부 ㅋ
