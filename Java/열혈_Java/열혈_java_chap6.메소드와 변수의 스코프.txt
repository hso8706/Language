*메소드와 변수의 스코프

-method

>특징
1. method 내의 문장은 순차적으로 실행된다.
2. 자바 프로그램은 main이라는 이름의 method에서부터 시작을 한다.
// 같은 class내에서 method가 있어야 묶어서 작동시킬 수 있음.
3. 괄호 내의 문장은 매개변수를 나타낸다. ( 예. String[ ] args)
// 매개변수에는 자료형까지 정확히 작성해줘야 한다.
4. 생김새와 접근법은 C언어의 함수와 비슷하다고 생각하면 될 듯.

>매개변수가 없는 method
: 매개변수가 없을 경우는 그냥 빈 괄호로 둔다. 접근시에도 빈괄호만 적어 실행시키면 된다.
>return값이 존재하는 method
: return값이 있는 함수랑 똑같~
: return이 없으면 void, 있으면 return 시킬 값의 자료형을 작성.
: return 은 값을 반환시키는 기능과 method를 종료하는 기능이 있음.

>변수의 스코프(scope)
: 변수의 접근 가능 영역, 즉 존재할 수 있는 영역을 의미
// 지역변수는 선언된 지역을 벗어나면 접근이 불가능할 뿐더러 메모리 공간에서 소멸됨.

>method 재귀 호출
: 기존에 알던대로, java에서도 가능함.
